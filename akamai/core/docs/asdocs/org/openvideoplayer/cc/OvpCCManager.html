Classpublic class OvpCCManager
InheritanceOvpCCManager OvpCuePointManager flash.events.EventDispatcher

The OvpCCManager class provides closed captioning for video playback and supports a subset of the W3C Timed Text Authoring Format 1.0 - Distribution Format Exchange Profile (DFXP). The DfxpParser class contains details about the supported subset.

See also
