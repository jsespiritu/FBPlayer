Packageorg.openvideoplayer.rss
Classpublic class RSSFilterFields
InheritanceRSSFilterFields Object

The FilterFields class is used by the filterItemList method in the XxxMediaRSS class. This class provides the caller of the filterItemList method with a simple way to specify which fields to search in a media RSS feed.

The Media RSS specification can be viewed at http://search.yahoo.com/mrss.
