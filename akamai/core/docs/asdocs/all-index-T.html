ThumbnailTO — class, package org.openvideoplayer.rss
The ThumbnailTO class is a transfer object defining the data representation of a thumbnail node in a media RSS feed. This class is used by the XxxMediaRSS class and is not invoked directly by end users. The properties in this class are defined by the Media RSS specification which can be viewed at http://search.yahoo.com/mrss.
