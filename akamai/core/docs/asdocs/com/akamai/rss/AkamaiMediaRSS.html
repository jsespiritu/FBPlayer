filterItemList()method
public function filterItemList(filterText:String, filterFields:RSSFilterFields = null, filterMatch:int, filterDate:Date = null, filterDateMatch:int):Array

Returns a filtered play list, allowing text filtering or date filtering. The properties in the FilterFields class allow you to specify which RSS fields will be used for the filtering. Calling this method with both filterText and filterDate arguments will essentially result in a double-filtering: the results of the text filter are then filtered again using the date filter. For date filtering the filterMatch argument is ignored.

Parameters

filterText:String — The text string to search for in the item list of RSS feeds.
 
filterFields:RSSFilterFields (default = null) — A FilterFields object specifying which fields in the item list of RSS feeds to search. If this argument is null, all fields will be searched.
 
filterMatch:int (default = NaN) — Either FILTER_ANY or FILTER_ALL. FILTER_ANY will find a match if any string contained in the filterText argument is found in any of the filterFields specified. FILTER_ALL will find a match only if the entire string is found in any of the filterFields specified.
 
filterDate:Date (default = null) — A date to search for in the item list of RSS feeds that contain dates. If the filterFields argument is null, all fields containing dates will be searched. In order for the 'less than or equal to' comparison to work as expected, we will set the time of this object to midnight if the hours, minutes, and seconds are all equal to zero. If this is not the desired behavior, set the seconds to 1 on the Date object before calling this method.
 
filterDateMatch:int (default = NaN) — Either FILTER_DATE_LEQ specifying 'less than or equal to', or FILTER_DATE_GEQ specifying 'greater than or equal to'.

Returns
Array — An array containing the filtered list or null if no matching items were found.
